## How to update
* chrome docker tag 선택
  * https://hub.docker.com/r/selenium/node-chrome/tags
* all_build.sh 에서 TAG 수정
  * github action 으로 image 가 생성됨
* 이미지 확인: https://hub.docker.com/r/anyonecantest/testcafe/tags
